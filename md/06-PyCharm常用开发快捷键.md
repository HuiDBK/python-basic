# PyCharm常用开发快捷键

## 前言

> 记住常用的开发快捷键，可以节省时间，提升开发效率。

<br/>

## 常用快捷键

| 快捷键             | 功能描述                                |
| ------------------ | --------------------------------------- |
| `Ctrl + alt + L`   | 格式化代码                              |
| `Shift + Enter`    | 在下方新建一行，光标移到新行行首        |
| `Ctrl + Enter`     | 在下方新建一行，光标不动                |
|                    |                                         |
| `Ctrl + D`         | 复制当前行到下一行                      |
| `Ctrl + X`         | 剪切当前行，可以当做删除行              |
| `Ctrl + /`         | 注释或取消注释                          |
|                    |                                         |
| `Ctrl Shift -`     | 收缩所有代码块                          |
| `Ctrl Shift +`     | 展开所有代码块                          |
| `Shift + Tab`      | 反向缩进代码                            |
|                    |                                         |
| `Ctrl + N`         | 查看所有类名称                          |
| `Ctrl + Shift + N` | 查找项目中的任何文件                    |
| `Shift + F10`      | 运行当前文件                            |
| `Alt + F12`        | 开启/关闭 PyCharm `Terminal` 终端控制台 |

<br/>

## 系统自带

| 快捷键        | 功能描述           |
| ------------- | ------------------ |
| `home`        | 光标跳转到行首     |
| `end`         | 光标跳转到行尾     |
| `Ctrl + home` | 光标跳转到文件开头 |
| `Ctrl + end`  | 光标跳转到文件末尾 |
| `Ctrl + F`    | 字符串匹配         |
